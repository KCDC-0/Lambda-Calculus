# Lambda-Calculus
Collection of mini projects exploring the mathematical logic and language design behind lambda calulus


## Equation Finder
Added to a repo "Equation-Finder": Returns all possible ways to concatenate, add or subtract the numbers 1 to 9 in order to get the given input, in a single line of code. The goal is to use as little characters in a single line of script as possible, allowing different in-built functions and imported modules

I used fixed point operators here to shortern the function that sllows for the in-built 'eval' function
