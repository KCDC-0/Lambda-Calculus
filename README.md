# Lambda-Calculus
Collection of mini projects exploring the mathematical logic and language design behind lambda calculus. These are mostly written in python, other languages may be added later. Most functions have the goal of being one-liners and are practices of 'code-golf'. Improvements to the functions may be added later on.

## Equation Finder
Added to a repo "Equation-Finder": Returns all possible ways to concatenate, add, or subtract the numbers 1 to 9 in order to get the given input, in a single line of code. The goal is to use as few characters in a single line of script as possible, allowing different in-built functions and imported modules

I used fixed point operators here to shorten the function that allows for the in-built 'eval' function
https://github.com/Bifameemo/equation-finder


## Prime Finder
Inefficient but logically satisfying way of looping through all numbers starting from 2 up to an ending number and keeping track of the primes


## Factorial
Recursive or iterative methods of returning a factorial of a given number. First and simplest project in this collection, done as an introduction to applying lambda calculus logic in a typed program


## Church Encodings
Implementation of church numerals, common logic gates, and basic arithmetic as python functions. These can be used as building blocks for longer lambda calculus functions
